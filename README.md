# Redesain Halaman Utama Website Fakultas Teknik UNRI

## Deskripsi Proyek
Proyek ini merupakan hasil modifikasi halaman utama website Fakultas Teknik Universitas Riau (FT UNRI) yang bertujuan untuk meningkatkan tampilan visual, responsivitas, serta pengalaman pengguna. Redesain dilakukan dengan menggunakan teknologi HTML, CSS, JavaScript, dan framework Bootstrap sebagai bagian dari materi praktikum front-end web. Website yang sebelumnya memiliki tampilan cukup padat dan kurang fleksibel di berbagai perangkat, diperbaiki menjadi lebih modern dan terstruktur. Seluruh perubahan dilakukan secara bertahap oleh anggota kelompok dengan memanfaatkan sistem version control GitHub. Setiap bagian yang dimodifikasi memiliki tujuan tertentu agar website menjadi lebih efektif dalam menyampaikan informasi. Dengan adanya redesain ini, diharapkan pengguna dapat mengakses website dengan lebih nyaman dan efisien.

## Teknologi yang Digunakan
1. HTML
2. CSS
3. JavaScript
4. Bootstrap

## Perubahan yang sudah dimodifikasi
Perubahan utama dalam proyek ini terletak pada peningkatan tampilan visual dan struktur halaman utama. Website dimodifikasi agar memiliki layout yang lebih rapi dan tidak terlalu padat seperti versi sebelumnya. Selain itu, diterapkan desain responsif agar tampilan dapat menyesuaikan berbagai ukuran layar. Penambahan efek interaktif juga dilakukan untuk meningkatkan pengalaman pengguna saat berinteraksi dengan website. Penggunaan Bootstrap membantu dalam mempercepat proses pengembangan serta menjaga konsistensi desain. Secara keseluruhan, proyek ini berfokus pada peningkatan kualitas tampilan dan kemudahan akses informasi.

## Bagian Modifikasi dan Alasan

### 1. Navbar
Bagian navbar dimodifikasi dengan menyederhanakan struktur menu agar lebih mudah digunakan oleh pengguna. Pada website asli, navigasi terlihat cukup kompleks sehingga dapat membingungkan pengguna baru. Pada hasil redesain, menu dibuat lebih ringkas dan jelas dengan tampilan yang lebih modern. Selain itu, navbar juga dibuat responsif sehingga dapat berubah menjadi menu toggle pada perangkat mobile. Perubahan ini bertujuan untuk meningkatkan kemudahan navigasi serta pengalaman pengguna dalam mengakses halaman.
### 2. Hero Section
Hero section dimodifikasi dengan menambahkan background image, overlay, serta teks utama yang lebih menonjol. Pada versi sebelumnya, bagian ini belum dimanfaatkan secara maksimal sebagai daya tarik utama halaman. Dengan penambahan elemen visual, hero section menjadi lebih menarik dan informatif. Posisi teks juga diatur agar lebih terpusat dan mudah dibaca oleh pengguna. Perubahan ini bertujuan untuk memberikan kesan pertama yang baik saat pengguna membuka website.
### 3. Konten Berita
Bagian berita diubah dari tampilan daftar menjadi tampilan berbasis card menggunakan sistem grid. Setiap berita dilengkapi dengan gambar, judul, dan deskripsi singkat agar lebih informatif. Tampilan card memberikan struktur yang lebih rapi dibandingkan tampilan sebelumnya. Selain itu, ditambahkan efek hover untuk meningkatkan interaksi pengguna. Perubahan ini bertujuan agar informasi lebih mudah dibaca dan terlihat lebih menarik secara visual.
### 4. Tampilan Responsif
Seluruh halaman dimodifikasi menggunakan Bootstrap agar dapat menyesuaikan ukuran layar secara otomatis. Pada versi sebelumnya, tampilan belum sepenuhnya optimal di perangkat mobile. Dengan sistem grid Bootstrap, layout dapat berubah sesuai ukuran layar pengguna. Elemen seperti gambar dan teks juga disesuaikan agar tetap proporsional. Perubahan ini bertujuan untuk memastikan website dapat diakses dengan nyaman di semua perangkat.

## Pembahasan Kode 

### 1. File index.html
File index.html terdapat struktur utama dari halaman website yang berisi susunan elemen seperti navbar, hero section, konten berita, dan footer. Pada proyek ini, file ini dimodifikasi dengan menyusun ulang layout agar lebih terstruktur dan mudah dipahami. Penggunaan elemen HTML dibuat lebih rapi dan mengikuti standar pengembangan web modern. Selain itu, integrasi Bootstrap dilakukan untuk mendukung tampilan responsif. Struktur grid digunakan untuk mengatur posisi elemen agar lebih konsisten. Dengan perubahan ini, halaman utama menjadi lebih terorganisir dan mudah dikembangkan.
### 2. File style.css
File style.css digunakan untuk mengatur tampilan visual seperti warna, font, dan layout. Pada proyek ini, CSS dimodifikasi untuk memberikan tampilan yang lebih modern dan menarik. Penyesuaian dilakukan pada bagian hero, card, dan elemen lainnya agar terlihat lebih rapi. Selain itu, ditambahkan efek hover untuk meningkatkan interaksi pengguna. Penggunaan spacing juga diperbaiki agar tampilan tidak terlalu padat. Dengan perubahan ini, tampilan website menjadi lebih nyaman dan profesional.
### 3. File script.js
File script.js digunakan untuk menambahkan interaksi pada website. Pada proyek ini, JavaScript digunakan untuk mendukung efek interaktif seperti animasi atau respon terhadap aksi pengguna. Script juga membantu meningkatkan pengalaman pengguna agar tidak hanya melihat tampilan statis. Struktur JavaScript dibuat terpisah agar kode lebih rapi dan mudah dikelola. Penggunaan JavaScript meskipun sederhana tetap memberikan nilai tambah pada website. Dengan adanya script ini, website menjadi lebih dinamis dan menarik.
### 4. Folder assets
Folder assets digunakan untuk menyimpan file pendukung seperti gambar yang digunakan dalam website. Pada proyek ini, gambar digunakan untuk memperkuat tampilan visual terutama pada hero dan berita. Pemilihan gambar disesuaikan dengan konteks Fakultas Teknik agar relevan. Selain itu, penyimpanan dalam folder khusus membuat struktur project lebih rapi. Pengelolaan aset yang baik juga memudahkan pengembangan ke depannya. Dengan adanya folder ini, semua media terorganisir dengan baik.
